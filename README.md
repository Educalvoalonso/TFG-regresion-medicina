# TFG-regresion-medicina
# Regresión lineal y logística: fundamentos teóricos y aplicaciones en medicina

Este repositorio contiene los programas en R utilizados en el Trabajo Fin de Grado
"Regresión lineal y logística: fundamentos teóricos y aplicaciones en medicina".

## Contenido

- `diabetes_regresion_lineal.R`: análisis de regresión lineal aplicado al conjunto de datos de diabetes.
- `cancer_mama_regresion_logistica.R`: análisis de regresión logística, validación cruzada y regularización aplicado al conjunto Breast Cancer Wisconsin.

## Librerías principales

Los scripts utilizan, entre otras, las siguientes librerías de R:

- `tidyverse`
- `ggplot2`
- `caret`
- `glmnet`
- `pROC`
- `corrplot`
- `mlbench`
- `pscl`

## Reproducibilidad

Los conjuntos de datos utilizados proceden de librerías públicas, por lo que no se incluyen datos personales ni información clínica identificable.
